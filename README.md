# Venetian Studio Template 01.1

Base estática para un estudio de arquitectura con enfoque en:

- conversión por WhatsApp
- SEO técnico y SEO local inicial para Ciudad de Guatemala
- velocidad y compatibilidad con GitHub Pages

## Archivos principales

- `index.html`
- `style.css`
- `script.js`
- `robots.txt`
- `sitemap.xml`
- `assets/images/`
- `assets/icons/`

## Publicación en GitHub Pages

1. Ir a `Settings > Pages`
2. Seleccionar `Deploy from a branch`
3. Elegir `main` y carpeta `/ (root)`
4. Guardar y esperar la primera publicación
